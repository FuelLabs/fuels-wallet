### Get Wallet

`getWallet(address: string | AbstractAddress): FuelWeb3Wallet`<br />
Return a FuelWeb3Wallet. This class extends the fuels-ts SDK's `WalletLocked`,
enabling all of the methods available for `WalletLocked` while using a FuelWeb3Provider on the connection point to request signed actions.

<CodeImport file="../../sdk/src/FuelWeb3.test.ts" testCase="getWallet" />

### Get Provider

`getProvider(): FuelWeb3Provider`<br />
Return a FuelWeb3Provider. This class extends the fuels-ts SDK's `Provider`,
enabling all of the methods available for `Provider` while using the FuelWeb3SDK on signature points to request user permissions.
