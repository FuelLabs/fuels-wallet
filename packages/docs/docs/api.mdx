### Get Wallet

`getWallet(address: string | AbstractAddress): FuelWalletLocked`<br />
Return a FuelWalletLocked. This class extends the fuels-ts SDK's `WalletLocked`,
enabling all of the methods available for `WalletLocked` while using a FuelWalletProvider on the connection point to request signed actions.

<CodeImport file="../../sdk/src/Fuel.test.ts" testCase="getWallet" />

### Get Provider

`getProvider(): FuelWalletProvider`<br />
Return a FuelWalletProvider. This class extends the fuels-ts SDK's `Provider`,
enabling all of the methods available for `Provider` while using the FuelSDK on signature points to request user permissions.
